Angular provides several ways to safely access and manipulate the DOM. While direct DOM manipulation should be avoided, Angular offers tools like ElementRef, Renderer2, ViewChild, and decorators for controlled DOM interactions.

ElementRef provides direct access to the native DOM element. Use with caution as it bypasses Angular's security:
  • Only when Renderer2 cannot accomplish the task
  • Third-party library integration requiring direct DOM access
  • Be aware of XSS vulnerabilities
  • Not compatible with server-side rendering

Renderer2 is the recommended way to manipulate the DOM. It provides a safe abstraction layer:
  • setStyle/removeStyle: Manipulate element styles
  • addClass/removeClass: Add or remove CSS classes
  • setAttribute/removeAttribute: Manage element attributes
  • createElement/createText: Create new elements
  • appendChild/removeChild: Modify DOM tree
  • listen: Attach event listeners
