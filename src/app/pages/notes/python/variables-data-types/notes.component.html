Variables and Data Types

Learn about Python variables, strings, numbers, booleans, and basic data types.

Beginner

Variables

Variables are containers for storing data values. Python has no command for declaring a variable.

Python 
# Creating variables
name = "John"
age = 25
height = 5.9
is_student = True

print(name, age, height, is_student)
Output 
John 25 5.9 True

Basic Data Types

Strings (str)

Text data enclosed in quotes

name = "Alice"
message = 'Hello World'
multiline = """This is a
multiline string"""

Numbers

Integers and floating-point numbers

age = 25          # int
price = 19.99     # float
complex_num = 3+4j # complex

Booleans (bool)

True or False values

is_active = True
is_complete = False
result = 5 > 3  # True

None Type

Represents absence of value

value = None
print(value)  # None

Type Checking

Use type() function to check the data type of a variable.

Python 
name = "Alice"
age = 25
height = 5.9
is_student = True

print(type(name))      # <class 'str'>
print(type(age))       # <class 'int'>
print(type(height))    # <class 'float'>
print(type(is_student)) # <class 'bool'>
Output 
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>

Advanced Examples

Type Conversion

Converting between different data types

# String to number
age_str = "25"
age_int = int(age_str)
age_float = float(age_str)

# Number to string
num = 42
num_str = str(num)

# Boolean conversion
print(bool(1))    # True
print(bool(0))    # False
print(bool(""))   # False
print(bool("hi")) # True
Output 
True
False
False
True

Multiple Assignment

Assign multiple variables in one line

# Multiple assignment
x, y, z = 1, 2, 3
name, age, city = "Alice", 25, "NYC"

# Swapping variables
a, b = 10, 20
a, b = b, a  # Now a=20, b=10

# Unpacking
coordinates = (3, 4)
x, y = coordinates

Variable Naming Rules

Best practices for naming variables in Python

# Valid variable names
user_name = "Alice"
age2 = 25
_private_var = "hidden"
PI = 3.14159

print(user_name, age2, _private_var, PI)
Output 
Alice 25 hidden 3.14159
