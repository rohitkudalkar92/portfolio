• Must start with letter (a-z, A-Z) or underscore (_)
  • Can contain letters, numbers, and underscores
  • Case-sensitive (age and Age are different)
  • Cannot use Python keywords (if, for, while, etc.)
  • Cannot contain spaces or special characters (@, #, $, %, etc.)
  • Should be descriptive and meaningful
  • name - starts with letter
  • user_age - uses underscore
  • _private - starts with underscore
  • firstName - camelCase (valid but not Python convention)
  • MAX_SIZE - constant naming
  • count2 - contains numbers
  • is_valid - descriptive boolean name
  • 2name - starts with number
  • user-age - contains hyphen
  • first name - contains space
  • user@email - contains special character
  • if - Python keyword
  • class - Python keyword
  • $price - starts with special character
  • SyntaxError: Python cannot parse the invalid syntax
  • NameError: Using undefined variables
  • Keywords: Reserved words cannot be used as variable names
  • Special characters: Only letters, numbers, and underscores allowed
  • Variables & Functions: snake_case
  • Constants: UPPER_CASE
  • Classes: PascalCase
  • Private: _leading_underscore
  • Boolean values: True, False (first letter must be capital)
  • None value: None (first letter must be capital)
  • Invalid: true, false, none (lowercase won't work)
