Escape sequences are special character combinations that represent characters that are difficult to type or display directly in strings. They start with a backslash (\) followed by a specific character.

  • \n - Creates a new line - print("Hello\nWorld")
  • \t - Adds a tab space - print("Name:\tJohn")
  • \\ - Displays a literal backslash - print("Path: C:\\Users")
  • \" - Displays a double quote - print("He said \"Hello\"")
  • \' - Displays a single quote - print('It\'s working')

The newline character creates a line break in your text output.

The tab character adds horizontal spacing, useful for formatting aligned text.

Use double backslashes to display a literal backslash character.
