• Precedence: Parentheses > Exponentiation > Multiplication/Division > Addition/Subtraction
  • Comparison chaining: a < b < c is equivalent to (a < b) and (b < c)
  • Logical short-circuit: 'and' stops at first False, 'or' stops at first True
  • Identity vs Equality: 'is' checks object identity, '==' checks value equality
  • Assignment operators: Modify variable in-place (+=, -=, *=, /=, etc.)
